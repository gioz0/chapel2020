## Declare Variables, Constants and Parameters
  declaration:
  - var identifier [:type] [=init_expr];
  - const identifier [:type] [=init_expr];
  - param identifier [:type] [=init_expr]; 



```chapel
var var1=33;
writeln(var1);
```
